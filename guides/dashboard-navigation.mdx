## Main Navigation Structure

### Sidebar Navigation Layout

<AccordionGroup>
  <Accordion title="Dashboard (Top Level)">
    **Primary Dashboard:**
    - **Location**: Top of sidebar navigation
    - **Icon**: Dashboard icon
    - **Purpose**: Financial overview and quick actions
    - **URL**: `/dashboard`
  </Accordion>
  
  <Accordion title="1. Inputs Section">
    **Data Sources:**
    - **Bank Transactions**: `/bank-transactions` - Import and categorize bank statements
    - **Email Processing**: `/email` - Process receipts and bills from emails
    - **Purpose**: Import financial data into the system
  </Accordion>
  
  <Accordion title="2. Personalize Section">
    **Customization:**
    - **Categories**: `/categories` - Organize transaction categories
    - **Custom Rules**: `/custom-rules` - Create automated rules for transaction processing
    - **Allocations**: `/allocations` - Set budget allocations and compare against spending
    - **Purpose**: Customize the system to your business needs
  </Accordion>
  
  <Accordion title="3. Analysis & Management Section">
    **Core Features:**
    - **All TXns**: `/all-transactions` - View and manage all transactions with AI tools
    - **Sub-items**:
      - **Expenses (One-time)**: `/expenses` - Manage one-time expense transactions
      - **Patterns (Recurring)**: `/bills` - Manage monthly recurring patterns and payments
      - **Travel Expenses**: `/travel-expenses` - Manage vehicle trips and travel deductions
  </Accordion>
  
  <Accordion title="4. Reports and Trends Section">
    **Analytics & Exports:**
    - **Insight+**: `/tax` - Analysis dashboard with variance, trends, and financial insights
    - **Tax Export**: `/ato-export` - Export transactions for tax reporting and deduction tracking
    - **AI Assistant**: `/ai` - AI-powered financial insights
  </Accordion>
</AccordionGroup>

## Navigation Methods

### Primary Navigation Paths

<Steps>
  <Step title="Sidebar Navigation">
    Use the main sidebar to access all features:
    1. Click hamburger menu (☰) on mobile
    2. Use expanded sidebar on desktop
    3. Navigate through numbered sections
    4. Click on feature names to access pages
  </Step>
  
  <Step title="Dashboard Quick Access">
    From the dashboard:
    1. Use quick action buttons for common tasks
    2. Click on widget elements to drill down
    3. Use "View All" links for detailed views
    4. Access recent items directly
  </Step>
  
  <Step title="Direct URL Navigation">
    For power users:
    1. Bookmark frequently used pages
    2. Use browser address bar for direct access
    3. Create custom shortcuts
    4. Use browser back/forward buttons
  </Step>
</Steps>

## Troubleshooting Navigation

### Common Navigation Issues

<AccordionGroup>
  <Accordion title="Sidebar Not Visible">
    **Mobile Issues:**
    - Tap hamburger menu (☰) to open sidebar
    - Swipe from left edge to open sidebar
    - Check if sidebar is collapsed (desktop)
    
    **Desktop Issues:**
    - Click expand button in collapsed sidebar
    - Check browser zoom level
    - Refresh page if sidebar is stuck
  </Accordion>
  
  <Accordion title="Page Not Loading">
    **Common Solutions:**
    - Check internet connection
    - Refresh browser page
    - Clear browser cache
    - Try different browser
    - Check URL for typos
  </Accordion>
</AccordionGroup>

### Performance Tips

<CardGroup cols={2}>
  <Card title="Browser Optimization" icon="globe">
    - Use modern browsers (Chrome, Firefox, Safari, Edge)
    - Enable JavaScript
    - Clear cache regularly
    - Disable unnecessary extensions
  </Card>
  
  <Card title="Network Optimization" icon="wifi">
    - Use stable internet connection
    - Avoid public WiFi for sensitive data
    - Check firewall settings
    - Use VPN if required
  </Card>
</CardGroup>
