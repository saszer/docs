## Category Management

Categories help you organize and track your financial transactions effectively. AI2Fin allows you to create custom categories, set tax deductibility, and organize your expenses for better financial management.

<Card title="Access Categories" icon="folder" href="https://app.ai2fin.com/categories" horizontal>

</Card>

## Category Features

### Creating and Managing Categories

<Steps>
  <Step title="Create New Category">
    1. Click "Create Category" button
    2. Fill in category details:
       - Category name
       - Category type (expense/income/transfer)
       - Color selection
       - Description (optional)
    3. Set tax deductibility if applicable
    4. Save category
  </Step>
  
  <Step title="Edit Existing Category">
    1. Click on category in the list
    2. Click edit button (pencil icon)
    3. Modify category details as needed
    4. Update tax settings if required
    5. Save changes
  </Step>
  
  <Step title="Delete Category">
    1. Select category to delete
    2. Click delete button (trash icon)
    3. Confirm deletion
    4. Handle existing transactions
  </Step>
</Steps>
