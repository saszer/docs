## Custom Rules Management

Custom Rules allow you to automate transaction processing and categorization based on your specific business needs. Create intelligent rules that automatically categorize transactions, detect patterns, and streamline your financial management.

</Card>

## Custom Rule Features

### Creating and Managing Rules

<Steps>
  <Step title="Create New Rule">
    1. Click "Create Rule" button
    2. Define rule conditions:
       - Transaction description contains
       - Amount is greater/less than
       - Merchant name matches
       - Date range conditions
    3. Set rule actions:
       - Assign category
       - Set tax deductibility
       - Add tags or notes
       - Mark as business expense
    4. Test and activate rule
  </Step>
  
  <Step title="Edit Existing Rule">
    1. Click on rule in the list
    2. Click edit button (pencil icon)
    3. Modify conditions or actions
    4. Test updated rule
    5. Save changes
  </Step>
  
  <Step title="Test Rule Effectiveness">
    1. Select rule to test
    2. Click test button
    3. Review test results
    4. Adjust rule if needed
    5. Activate when satisfied
  </Step>
</Steps>

### Rule Types and Conditions

<AccordionGroup>
  <Accordion title="Condition Types">
    **Available Conditions:**
    - **Description Contains**: Text matching in transaction description
    - **Amount Range**: Transaction amount within specified range
    - **Merchant Match**: Specific merchant or vendor names
    - **Date Range**: Transactions within date periods
    - **Account Type**: Specific account or account types
    
    **Advanced Conditions:**
    - **Pattern Matching**: Regular expressions for complex patterns
    - **Multiple Conditions**: AND/OR logic combinations
    - **Frequency Rules**: Recurring transaction detection
    - **Context Rules**: Business vs personal classification
  </Accordion>
  
  <Accordion title="Action Types">
    **Available Actions:**
    - **Category Assignment**: Automatically assign categories
    - **Tax Settings**: Set tax deductibility and business use
    - **Tagging**: Add custom tags to transactions
    - **Notes**: Add automatic notes or descriptions
    - **Status Changes**: Mark transactions as reviewed/processed
    
    **Advanced Actions:**
    - **Amount Modifications**: Adjust amounts for calculations
    - **Date Adjustments**: Modify transaction dates
    - **Splitting**: Split transactions into multiple entries
    - **Merging**: Combine related transactions
  </Accordion>
</AccordionGroup>
