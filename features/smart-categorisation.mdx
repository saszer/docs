## How It Works

### AI Processing Pipeline

<Steps>
  <Step title="Transaction Analysis">
    AI analyzes each transaction including:
    - Merchant name and description
    - Transaction amount and timing
    - Payment method and location
    - Historical patterns
  </Step>
  
  <Step title="Pattern Recognition">
    Machine learning identifies:
    - Similar transaction types
    - Recurring merchant patterns
    - Spending behavior trends
    - Category associations
  </Step>
  
  <Step title="Confidence Scoring">
    AI provides confidence levels:
    - High confidence (90%+): Auto-categorized
    - Medium confidence (70-90%): Suggested category
    - Low confidence (<70%): Requires review
  </Step>
  
  <Step title="Learning & Improvement">
    System learns from:
    - User feedback and corrections
    - Accepted suggestions
    - Manual categorization choices
    - Business-specific patterns
  </Step>
</Steps>

### Categorization Methods

<AccordionGroup>
  <Accordion title="Merchant-Based Categorization">
    **AI Recognition:**
    - Identifies merchant names from transaction descriptions
    - Maps merchants to appropriate categories
    - Handles variations in merchant naming
    
    **Examples:**
    - "WOOLWORTHS" → Groceries
    - "OFFICEWORKS" → Office Supplies
    - "UBER TRIP" → Travel & Transport
    - "ADOBE SYSTEMS" → Software & Subscriptions
  </Accordion>
  
  <Accordion title="Description Analysis">
    **Natural Language Processing:**
    - Parses transaction descriptions
    - Extracts key terms and context
    - Identifies transaction purpose
    
    **Pattern Matching:**
    - Recognizes common business terms
    - Identifies expense types
    - Distinguishes business vs personal
  </Accordion>
  
  <Accordion title="Amount-Based Logic">
    **Transaction Analysis:**
    - Amount ranges for different expense types
    - Round number patterns (subscriptions)
    - Frequency analysis (recurring bills)
    
    **Smart Suggestions:**
    - High-value transactions → Equipment/Assets
    - Regular amounts → Subscriptions/Bills
    - Variable amounts → Variable expenses
  </Accordion>
</AccordionGroup>
