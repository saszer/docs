## How It Works

### Connect ChatGPT via OpenAI Apps SDK

Connect your ChatGPT to AI2Fin using OpenAI's Apps SDK. Once connected, you can ask ChatGPT questions about your finances:

**Example Conversation:**
```
You: "What are my tax deductible expenses this year?"
ChatGPT: "You have $12,450 in tax deductible expenses across 8 categories. 
         Your top deduction categories are:
         - Office Supplies: $3,245
         - Travel: $2,187
         - Professional Services: $1,890
         ..."
```

ChatGPT uses AI2Fin's MCP server to access your financial data and provide accurate, real-time answers about your expenses, bills, and tax deductions.


### How to Connect

1. **In your AI application** (ChatGPT, Claude, Cursor, etc.), add a new MCP server connection
2. **Use the URL:** `https://docs.ai2fin.com/mcp`
3. **Authenticate** using your API key (same as for the financial data MCP server)
4. **Start searching** - Your AI assistant can now search across all AI2Fin documentation

### Available Documentation Tools

- **SearchAi2FinDocumentation** - Search across the AI2Fin Documentation knowledge base to find relevant information, code examples, API references, and guides. Use this tool when you need to answer questions about AI2Fin Documentation, find specific documentation, understand how features work, or locate implementation details. The search returns contextual content with titles and direct links to the documentation pages.

The documentation MCP server is separate from the financial data MCP server. Use `https://docs.ai2fin.com/mcp` for documentation search, and `https://api.ai2fin.com/mcp` for accessing your financial data.
</Tip>

## Need Help?

<AccordionGroup>
  <Accordion title="Can't Connect to My Financial Data" icon="key">
    If you're having trouble connecting:
    
    - Make sure you're logged into your AI2Fin account
    - Check that your authentication token is valid
    - Try re-authenticating if your token has expired
    - For API keys, verify the key hasn't been revoked
  </Accordion>
  
  <Accordion title="Getting Rate Limit Errors" icon="gauge">
    If you see rate limit messages:
    
    - Wait a few minutes and try again
    - You can make up to 200 requests per 15 minutes
    - For heavy usage, consider upgrading your plan
  </Accordion>
  
  <Accordion title="No Results Found" icon="search">
    If you're not seeing expected data:
    
    - Check that you have transactions in the date range you're asking about
    - Try asking about a broader time period
    - Make sure your filters aren't too restrictive
    - Verify you have data imported into AI2Fin
  </Accordion>
  
  <Accordion title="Slow Responses" icon="clock">
    If responses are taking too long:
    
    - Try asking for summaries instead of all transactions
    - Narrow your date range
    - Ask more specific questions
    - Check your internet connection
  </Accordion>
</AccordionGroup>
